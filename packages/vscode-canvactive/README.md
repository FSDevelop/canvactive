# Canvactive VS Code Extension

Syntax highlighting for `.can` files.

The grammar treats Canvactive files as HTML-like component files and embeds JavaScript highlighting inside both `<script>` and `<render>` blocks.

## Local Development

Open this folder in VS Code and press `F5` to launch an Extension Development Host.

To package it later:

```bash
npx @vscode/vsce package
```
